# ONT-seq-data-analysis-epi-AD
This pipeline uses various software tools in command line as well as R to preprocess, quality control, and analyze data generated by the ONT RRMS promethion. 
Dorado software in command line is used to perform rebasecalling, barcode trimming and demultiplexing, and alignment, generating aligned BAM files. 
These BAM files are indexed and filter for read length and mapping quality using samtools. Mosdepth software and Qualimap are used to determine the coverage across the region of interest. 
Bcftools is used to obtain VCF files (SNPs). Following filter step, the Modkit software is used to determine the number of modification calls.

Quality control pipeline in R includes checking for batch effects across flow cells, outlier detection, min coverage threshold, sex prediction, PCA and density plots etc. 
Differential (hydroxy)methylation analysis is performed using the Limma package in R.
